We learned how to connect PHP to a MySQL/MariaDB database using PDO.

2ï¸âƒ£ try / catch and Exceptions

We used:

try {
   // code
} catch (PDOException $e) {
   echo $e->getMessage();
}

How it works:

try contains code that might fail.

If an error happens, PHP throws an Exception.

catch captures the exception.

$e->getMessage() returns the error message.

Execution stops inside try when throw happens and jumps to catch.

3ï¸âƒ£ Executing a SELECT Query
Method 1: query()
$stmt = $pdo->query("SELECT * FROM users");


Use query() when:

The SQL statement is simple.

There is no user input.

Method 2: prepare() + execute()
$stmt = $pdo->prepare("SELECT * FROM users WHERE id > :min_id");
$stmt->execute(['min_id' => 1]);


Use this method when:

There is user input.

You need security (prevents SQL injection).

Steps:

prepare() prepares the SQL statement.

:min_id is a placeholder.

execute() sends real values to placeholders.

4ï¸âƒ£ Retrieving Data

After executing the query, we get results using:

fetch()

Returns one row at a time.

while($row = $stmt->fetch()) {
   echo $row['name'];
}


Used with while loop.

fetchAll()

Returns all rows at once as an array.

$rows = $stmt->fetchAll();

foreach($rows as $row) {
   echo $row['name'];
}


Used with foreach loop.

FETCH_OBJ
