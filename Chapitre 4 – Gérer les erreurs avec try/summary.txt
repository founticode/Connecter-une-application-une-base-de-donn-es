ðŸ”¹ 1. Try / Catch Blocks

The try block contains the code that may produce an error (like SQL queries).

If an error occurs, the catch block handles it safely.

Example:

try {
    // risky code
} catch (PDOException $e) {
    echo "Error: " . $e->getMessage();
}


This prevents the application from crashing and allows controlled error handling.

ðŸ”¹ 2. PDOException

PDOException is a special type of error thrown by PDO when something goes wrong (SQL syntax error, connection error, etc.).

It allows us to catch and manage database-related errors properly.
